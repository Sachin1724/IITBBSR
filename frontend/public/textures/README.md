# Earth Textures

This directory contains texture maps for the realistic Earth model.

## Required Textures

Download these free NASA Earth textures and place them in this folder:

### 1. **earth_day.jpg** (Color/Albedo Map)
- Download from: https://www.solarsystemscope.com/textures/
- Or: https://planetpixelemporium.com/earth.html
- Resolution: 2K (2048x1024) or 4K (4096x2048)
- Shows continents, oceans, and clouds

### 2. **earth_normal.jpg** (Normal Map)
- Shows surface elevation details
- Makes mountains and terrain look 3D
- Same resolution as color map

### 3. **earth_specular.jpg** (Specular/Roughness Map)
- Shows which areas are shiny (water) vs rough (land)
- Black = rough (land), White = shiny (water)
- Same resolution as color map

## Quick Download Links

**Free NASA Visible Earth Textures:**
- https://visibleearth.nasa.gov/collection/1484/blue-marble
- https://www.solarsystemscope.com/textures/ (Free for non-commercial)

## File Naming

Make sure files are named exactly:
- `earth_day.jpg`
- `earth_normal.jpg`
- `earth_specular.jpg`

## Alternative: Use Procedural Earth

If you don't want to download textures, the component will show a blue procedural Earth as fallback.
